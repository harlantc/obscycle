Chandra Proposal Search Help

Cycle

The Call for Proposal (CfP) cycle of which the observation was a part.

Data Entry Information: See Multiple Selection Lists for more information on using lists

Proposal Number

8-digit number identifying the proposal for an observation

Data Entry Information: Any Proposal Number that contains the entered value will be returned. For example, entering 520 could match: 

  04800520
  05200866
  15200174

Type

The Proposal Type, options are:
  • GO: General Observer proposals
  • LP: Large Project and Very Large Project proposals
  • VLP: Very Large Project proposals (Cycle 5-12)
  • XVP: X-ray Visionary Project proposals (Cycle 13 and greater)
  • TOO: Target Of Opportunity
  • DDT: Director's Discretionary Time
  • GTO: Guaranteed Time Observation
  • CCT: Chandra Cool Targets
  • CAL: Calibration
  • Archive: Archive proposals
  • Theory: Theory proposals
  • CECS: Cold External Calibration Source proposals

P.I. Last Name

Last name of the Principal Investigator

Data Entry Information: The matching is case insensitive. Any PI Name that contains the entered value will be returned. For example, entering smith could match: 

    SMITH
    SMITHERS
    BLACKSMITH

Joint

Other observatories that were approved as part of this observation (if any). The Joint Observatory options are:
  • None: No joint time awarded for any other observatory.
  • HST: Hubble Space Telescope (HST) Observatory. This includes proposals approved at the Chandra Review (HST) and those approved at the HST Review(CXO-HST).
  • NOAO: National Optical Astronomy Observatory (NOAO) Observatory
  • NRAO: National Radio Astronomy Observatory (NRAO) Observatory
  • NuSTAR: NuSTAR Observatory
  • RXTE: Rossi X-ray Timing Explorer Mission
  • Spitzer: Spitzer Space Telescope Observatory. This includes proposals approved at the Chandra Review (Spitzer) and those approved at the Spitzer Review(CXO-Spitzer).
  • Suzaku: Suzaku Observatory
  • Swift: Swift Observatory
  • XMM: XMM-Newton Observatory. This includes proposals approved at the Chandra Review (XMM) and those approved at the XMM Review(CXO-XMM).

Sort by

The order in which the search results are displayed. The user may choose to sort by any element in the selection list.

Display

  • Format
    The two choices for the output display format are:
    • HTML: Proceeds to Search Results HTML page
    • Text: Returns text page of tab-separated results which can be printed or saved in a file

  • Row Limit
    Choose the maximum number of rows to display. The recommended maximum number is 100. If "No Limit" is selected and the search criteria do not limit the results sufficiently, the user may experience an unacceptable delay.

Multiple Selection Lists

  • The user may select none, any or all of the values in the list
  • Selecting none of the values in the list is the equivalent of selecting all of the values in the list (i.e. not selecting anything will cause the search to ignore that field, which is the same as using the field in the search with all possible values selected)
  • It is browser dependent how the user unselects a single value (so that no values are selected after some were previously selected)
  • To unselect a single value in most browsers, hold the control key down while clicking the mouse
