8.0 CAVEATS

- Works best with Mozilla Firefox(v1.5.0.5 or newer), SeaMonkey(v1.1.8 or newer) and Safari(v3.1.2 or newer) browsers.
- Accuracy of mouse pointer display is 1 pixel which may translate to significant number in data units depending on axis limits set.
- On Windows, you might have to configure your mouse pointer 'precision select' cursor to 'crosshair' in Controll Panel/Mouse Properties if it is does not show up when hoovering over the plot. 
- There are issues with PRoVis and Internet Explorer.
- There is a bug in gv using Ghostscript v8.51 in x11 device on SOLARIS platform. It has been fixed in Ghostscript v 8.63. This bug might manifest itself when PRoVis generated postscript is viewed. If user has gv with ghostscript version older than 8.63 configured as postscript viewer application in web browser then there are a number of ways to work around it: use ggv, setup gv to use x11alpha device, use some other postscript viewer.
