Chandra Status List Help Page

The status list displays observations that match the criteria of the specified report type.

Report TypeDescription
ScheduledObservations with status scheduled but no processing information received yet.
AP/no L2Observations with status scheduled but no level 2 product has been processed yet
Awaiting IngestLevel 2 products that have incomplete archive ingest
Awaiting V&VLevel 2 product has no completed V&V report. If a later level 2 product has a V&V report, then the observation is not considered awaiting V&V.
Awaiting CDO ReviewV&V has requested a CDO review which has not yet been completed
Awaiting Aspect ReviewV&V has requested an Aspect review which has not yet been completed
Awaiting DistributionLevel 2 product with V&V state of OK or Repr+DD, but has no distribution date.
SAPObservations awaiting, or in, Special Automated Processing (SAP); highest revision of a Level 2 product and V&V state of Repr+DD or ReprNoDD

Not all fields are displayed for all reports.
Field NameDescription
ObsId Observation Id number (assigned by CXCDS)
Obs Date Start date and time of observation
L2 Version Level 2 version of processing of observation
L2 AP Date Date of level 2 processing operation
Good Time Good data exposure time (ks)
Charge Time Time charged against allocation
V&V State Verification & Validation state of Automated Processing data products:
NULL: No V&V Report available yet
OK: Approved for distribution
Hold: Internal DS hold
Problem: Referred to CDO
Repr+DD: Reprocessing required but distribution approved
ReprNoDD:Reprocessing required; do not distribute
V&V Version Version of Verification & Validation of processing
V&V Date Date of Verification & Validation. Creation date for those awaiting V&V, CDO review or Aspect Review. Completion date for those awaiting data distribution or SAP.
Data Distr Date Data delivery date: Date PI was notified of data being available
