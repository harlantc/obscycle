Chandra Single Observation Thread Help Page

Obi Information

The Obi information for all Default entries are displayed for the current observation. If no default data is available for an Obi, then the latest revision is displayed for that obi. Good time is only displayed for Default entries.

Field NameDescription
Exp CycleExposure cycle (ACIS alternating exposure mode):
P=Primary, S=Secondary
ObiObservation interval number
Start TimeStart date and time of observation interval (Obi)
Stop TimeEnd date and time of observation interval (Obi)
Good TimeGood data exposure time (ks) (Default entries only.)
Issues Issues that apply to processing of observation interval(obi).Issues displayed in parentheses() have been closed.

Processing Information

All Science L2 information is displayed for all Obi's for the current observation ordered by Version. In addition, all completed Custom Processing and the latest uncompleted thread, if any, is displayed. The current default is highlighted.

Field NameDescription
Exp CycleExposure cycle (ACIS alternating exposure mode)
Proc Version Version of processing of observation (highest level, usually L2)
Proc Type Processing Type: Automated processing, custom processing, reprocessing
Proc Date Date of processing operation (highest level, usually L2)
Proc Status Processing status/uncompleted thread
Charge Time Time charged against allocation (ks)
Good Time Good data exposure time (ks)
Release Version of CXCDS software release used for processing (highest level, usually L2)
V&V Date Date of Verification & Validation
V&V Status Verification & Validation status of Automated Processing data products.
NULL: No V&V Report available yet
OK: Approved for distribution
Hold: Internal DS hold
Problem: Referred to CDO
Repro+DD: Reprocessing required but distribution approved
ReprNoDD: Reprocessing required; do not distribute
Review Review by Chandra Director's Office or Aspect team required
V&V Version Version of Verification & Validation of processing
Data Distr Date Data delivery date: Date PI was notified of data being available
Data Mail Date Date hard medium was mailed
