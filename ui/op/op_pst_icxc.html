Standard Status Tables The user has a choice of four standard sets of fields (Short, Long, Summary, Dates). Output format choices are regular HTML, tab-delimited, ASCII and RDB. For an explanation of the status fields see the Chandra Observations Processing Status Fields page.
Customized Status Tables These tables offer the same selection, ordering, and output format options as the Standard Status Tables, but allows the user to select the fields to be included.
Issue Tables The user has a choice of two standard sets of fields (Short and Long). Output format is a regular HTML table. For an explanation of the issue fields see the Chandra Observations Processing Issue Fields page.
