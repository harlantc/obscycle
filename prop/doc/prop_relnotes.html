Release R3A12.1

Beta Test version residing on ASCDS server; released for internal review on 05/02/97

Upgrades provided in Release R3A12.1 are summarized below for each application.

PRECESS GUI

  • fixed +/- parsing error
  • added description of acceptable format to error message
  • added access to a quick help file

COLDEN GUI

  • fixed warning that environment variable NH base was not set
  • added description of acceptable format to error message

PIMMS GUI

  • fixed use of input and output energy range values
  • added default energy range button to provide user with default ranges for selected instrument
  • formatted output and added button to allow viewing of verbose output
  • coded and added pileup calculation
  • added formula indicating how photon index is defined
  • added list of acceptable AXAF instrument/grating/filter combinations
  • added support for new heg1meg1 case
  • added support for new hesf case
  • removed support for 3 "all" cases

PIMMS ALGORITHM

  • extensively revised effective area curves (Belinda Wilkes); latest version is in /data/UserSupport1/master_cal/pimms_data_new; note that only cases supported in GUI are approved for use

DATES GUI

  • improved description of acceptable formats
  • added access to a quick help file

Release R3A12.4

Alpha Test of GO version residing on **TBD**; released for review on **TBD**

Upgrades provided in Release R3A12.4 are summarized below for each application.

OVERALL TOOLKIT

  • Toolkit GUI was redesigned and placed in a separate, resizable frame launched from the Toolkit Web Page. This allows users to resize the application to account for the different font styles and sizes found on different machines.
  • A Toolkit Control Panel was added to the Toolkit Web Page. The Control Panel can launch a Toolkit window as a separate process or thread from the Web browser window. This improves run time efficiency and also allows users to view other Web pages while using the toolkit. When not in use, the user can quit the Toolkit window to minimize CPU load on their machine. A new Toolkit can be launched by returning to the Toolkit Web Page and selecting "Start New Toolkit" on the Control Panel. The Control Panel will only allow one Toolkit window at a time.
  • A status bar on each tool provides the user with status while the tool is communicating with the server.
  • added additional text directing user to other documents
  • made all help files available from top of Toolkit (top of Java Applet Window frame)
  • changed references from "Science Instrument Notebook" to "Observatory Guide"

COLDEN GUI & ALGORITHM

  • upgraded to new version of Colden algorithm, which has been alpha tested on a PC w/LINUX for increased portability
  • now provides choice of Bell (Stark et al.) velocity-resolved data covering northern sky to 40 degrees south, or NRAO (Dickey & Lockman) data, not velocity-resolved, giving all-sky coverage
  • now uses FITS data files, clearing the way for increased portability

PIMMS GUI

  • for Raymond-Smith model, added display of actual "kT" value used (gridded value nearest to requested value is used)

PIMMS ALGORITHM

  • officially incorporated revised effective area curve for acis-i
  • upgraded to PIMMS v2.4; includes fix to problem preventing long path/file names, clearing the way for simplifying data directory organization
