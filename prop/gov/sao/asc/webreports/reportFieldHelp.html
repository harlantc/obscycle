Proposal Information

  • Subject Category: The subject category for the proposal.

  • Proposal Title: The title of the proposal.

  • Type: The type of proposal submission (e.g., GO, THEORY, ARCHIVE, LP, VLP, TOO)

  • Constrained Targets: This field is set to "Yes" if any target in the proposal has required constraints. If no required constraints exist but at least one preferred constraint exists, it is set to "Preferred". If no constraints exist, it is set to "No".

  • View Associated File: This field will display the associated proposal files for this proposal. Just select the desired file and it will be displayed in a separate window. The options are : Proposal Conflicts, Science Justification, RPS Parameters, and optionally any proposer input associated with the proposal. Separate links are also provided for any review reports or technical report associated with the proposal.

Fields

For the "Degree of effort" field, select the option that best applies when reviewing this proposal.
  • Degree of effort required to achieve analysis goals (for Phase 2 review): Select the degree of effort that will be required to analyze the Chandra data and accomplish the science goal. Options are Easy, Average, Above-average. Required.
