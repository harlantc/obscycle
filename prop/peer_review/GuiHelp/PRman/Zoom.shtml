Panel Zoom

The Panel Zoom feature applies only to the Panel GUI window where the data is viewed and to the Status Page window. There are three forms of zooming:

Zoom in
Increase the font size in the window
Zoom out
Reduce the font size in the window
Zoom normal
Resume default font size in the window
